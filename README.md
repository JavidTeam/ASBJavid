# Installation

```sh
# Let's install the bot.
cd $HOME
git clone https://github.com/JavidTeam/ASBJavid.git
cd ASBJavid
chmod +x Javid.sh
./Javid.sh install
./Javid.sh 
# Enter a phone number & confirmation code.

# For Auto Launch:
cd ASBJavid
chmod 777 autobd.sh
screen ./autobd.sh
```
